## Installation

Add `najs-binding`, `najs-eloquent`, `najs-eloquent-mongodb`

```bash
yarn add najs-binding najs-eloquent najs-eloquent-mongodb
```

or

```bash
npm install najs-binding najs-eloquent najs-eloquent-mongodb
```

That's it.
